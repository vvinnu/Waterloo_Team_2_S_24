# Home Service web application

## Overview
The home service App allows users to conveniently book and manage appliance repair and maintenance services. It connects with various external systems and provides features like searching for and booking appointments, tracking technician progress, communicating with technicians, viewing service history, and more.
  

## Setup and Installation
### Prerequisites
Tools Required
- Node.js
- npm 
- MySQL
IDE Required
- Visual Studio Code

### Installation
1. **Clone the repository:**
    ```sh
    git clone https://github.com/vvinnu/Waterloo_Team_2_S_24.git
    ```

2. **Install dependencies:** (if not installed)

    npm install
   
    npm install express 

### Running the Application

  Start the MySQL server
  
  Run the home_services.sql file in mysql server to create the required database and tables
  
  Navigate to \final_home_service\src\connection folder
  
  run the command "node server.js" to connect the the SQL server
  
  Navigate to the root folder \final_home_service
  
  Run the command "npm start"

Repeat the same process for the /admin/ folder to run the admin web page
